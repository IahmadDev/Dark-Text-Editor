# Dark Text Editor

Dark text editor is a windows application build in c# which can perform following actions

## File Menu
New, Open, Save, Save As, Close
## Edit Menu
Copy, Cut, Paste
## Font Menu
Bold, Underline, Strikethrough, Italic, Font Menu
